> Interactively pick and run a NPM script.


## Install

```
$ npm install --global xqt
```


## Usage

Pick and run a script from the package.json file
```
[~/package]$ xqt
? Pick a script …
▸ dev
  test
  build
```

Shows xqt help information.
```
[~/package]$ xqt --help
```

Shows xqt version.
```
[~/package]$ xqt --version
```
